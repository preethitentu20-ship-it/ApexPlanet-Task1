# ApexPlanet Internship - Task 1

I am excited to share that I have successfully completed **Task 1** of my Web Development Internship at ApexPlanet.

## Project Overview
This project is a basic webpage built using **HTML, CSS, and JavaScript**.  
It demonstrates:
- Logos of **Google, YouTube, and Instagram** displayed neatly in a single row across the page.
- Navigation links directly below each logo to access the actual websites.
- A creative JavaScript button that shows a friendly alert messages.

## Technologies Used
- **HTML** → for webpage structure  
- **CSS (Flexbox)** → for layout and styling  
- **JavaScript** → for interactivity  

## Learning Outcome
Through this task, I strengthened my skills in structuring webpages, styling layouts with Flexbox, and adding interactivity with JavaScript. This internship is helping me gain **hands‑on experience in web development** and improve my ability to build clean, interactive, and professional webpages. 🚀
